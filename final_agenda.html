Lovespace error: "Error retrieving credentials"

Posted: 13 October 2023 10:27

Some users might see an error message pop up when trying to access a course offering in Brightspace stating "Error retrieving credentials".

Start: Wednesday 11 October 2023 9:00

Status: The supplier has investigated this issue and states that the error can be resolved by clearing the cache and cookies of your browser.

Affected Users: Users will see an error message pop up when trying to access a course offering in Brightspace stating "Error retrieving credentials". Users will be able to use Brightspace without any issues after dismissing the error message, but the error message will show up again when accessing another course offering.

All users are advised to clear cookies and cache in case they still experience issues using Brightspace.

We Apologize for any inconvenience caused.
